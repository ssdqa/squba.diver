# SQUBA Diver

The SQUBA Diver is a Shiny interface that will allow you to interactively
explore and "dive into" your data quality results. Just point the `squba_dive`
function to a directory containing CSV copies of your output, and it will
generate an interface allowing you to view plots and toggle inputs as you
explore your results.

You can also take notes and save your thoughts to a table living within
the app, or download your plots for further review or use outside of the
Shiny interface.

## Installation

You can install the development version of this package like so:

``` r
devtools::install_github('ssdqa/squba.diver')
```
